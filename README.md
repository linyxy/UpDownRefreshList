UpDownRefreshList 下拉刷新，上拉刷新表格
=================

This is a expanded list with two extra features

1. when the list is drag down , it would auto refresh
2. when the list is lift to the end, it would load more content

1. 列表涵盖下拉刷新
2. 列表支持下拉加载更多
